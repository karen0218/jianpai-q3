Day 06-08 · 低保真Flow、页面结构与关键文案

本文件是可以进入原型制作的结果内容：每条 Flow 给出完整页面节点、用户看到什么、点击什么、系统反馈什么；同时明确页面清单、关键文案和 Demo 原型保真度。

本阶段定稿结论 低保真阶段不追求视觉细节，重点把路径跑通：学生从哪里开始、看到什么、点哪里、系统如何回应、如何进入下一步。Flow A 已升级为 Pad 首页启动中枢，必须验证“主行动清楚 + 首页内容足够丰富”；Flow B 必须完整可测，Flow C 可做轻闭环。
Flow A 必做 / 高保真候选

Pad首页中枢 → 今日状态主卡 → 情绪打卡 → AI接住 → 今日推荐/功能分流 → 今日小行动 → 完成反馈 → 首页回流

页面节点说明

节点 用户看到什么 用户点什么 系统反馈什么 保真度
A1 Pad首页中枢 横向 Pad 布局：今日状态主卡、AI伙伴侧栏、今日推荐区、成长反馈条、次级功能入口。主行动仍是“今天感觉怎么样？”。 点击主卡“开始记录”或直接点一个状态词；也可从右侧 AI 伙伴进入 Flow B。 进入状态选择；首页其他区域保持可见但不抢主行动。 高保真
A2 状态选择 6-12 个状态词：还好、有点烦、累、紧张、开心、说不清。 选择状态词；可跳过补充说明。 选中状态高亮，出现“要不要再补一句？” 高保真
A3 打卡确认 “今天的状态：有点烦”，可写一句“可能是作业太多”。 点击“就这样记录”。 状态被保存，进入 AI 接住卡。 中高保真
A4 AI接住 小星/吉塔说一句回应，先接住感受，不急着分析。 点击“看看今天可以做什么”。 根据状态推荐小行动。 高保真
A5 今日推荐分流 一个主推荐，例如“2分钟呼吸放松”，旁边保留 1-2 个次级入口：脑之岛轻训练、继续找 AI 聊。 点击主推荐“开始”，或点次级入口。 主推荐进入小行动；次级入口进入对应假跳转或 Flow B，不展开功能列表。 高保真
A6 行动进行 简单进度、温和引导，不出现复杂功能。 点击完成，或等待进度结束。 显示完成反馈。 中保真
A7 完成反馈 “今天你完成了一次小小的照顾”，记录到成长本。 点击“回首页”或“继续聊聊”。 首页状态更新；也可进入 Flow B。 高保真
A8 首页回流 首页主卡变成“今天已记录”；AI伙伴显示一句延续反馈；今日推荐变成待完成/已完成状态；成长反馈条新增一小步。 点击“继续和小星聊”、查看成长记录，或浏览次级模块。 证明产品记得刚才的状态和行为，首页从入口页变成状态中枢。 高保真

Flow A 页面清单

页面 首页怎么改 / 情绪打卡怎么走 / AI怎么接 / Pad空间怎么用 是否必须做
Pad首页中枢从功能平铺改为“今日状态主行动 + AI伙伴侧栏 + 今日推荐区 + 成长反馈条 + 次级功能入口”；丰富但不抢主线。必须
情绪状态选择低压力词库 + 可跳过文字输入；先选状态，再决定是否补充。必须
AI接住卡小星/吉塔复述状态、表达陪伴、给出下一步入口。必须
今日推荐分流基于状态推荐一个主动作，同时露出 1-2 个相关次级入口，不展开大菜单。必须
行动完成反馈反馈今天的一小步，并提示已记录。必须
首页回流状态首页从“开始打卡”变成“今天已记录”；AI、推荐、成长区域同步更新。必须

Pad 首页五区低保真结构

区域 用户看到什么 交互规则 保真度
今日状态主卡“今天先从哪里开始？”+ 情绪快捷词 + 开始记录按钮。首屏最大区域，承接 Flow A 第一点击。高保真
AI伙伴侧栏小星/吉塔/罗素今日一句话，根据是否打卡切换。可进入 Flow B，但视觉权重低于主卡。高保真
今日推荐区1 个主推荐 + 1-2 个次级推荐，推荐理由来自当前状态。只显示相关下一步，不展示完整功能库。高保真
成长反馈条今日已记录、已完成、获得星星或成长一小步。完成后更新，用来验证模块之间是否连起来。中高保真
次级功能入口脑之岛、课程、测评、信箱、个人中心等入口。可见但弱化，用于体现 Pad 信息量，不承担主流程。中保真/假跳转

页面节点说明

节点 用户看到什么 用户点什么 系统反馈什么 保真度
B1 全局AI入口 页面右下/底部稳定入口，文案随场景变化。 点击“找小星聊聊”或“让吉塔帮我缓一下”。 进入伙伴选择或直接进入对应角色。 高保真
B2 三IP选择 不是角色设定说明，而是“想被陪陪/想先稳住/想拆问题”。 选择小星、吉塔或罗素。 进入对应对话开场；可随时切换。 高保真
B3 对话开场 预设话题 + 输入框，例如“学习有点累”“想不清楚”“只是想聊聊”。 点预设话题或输入一句。 AI开始 2-3 轮脚本化回应。 高保真
B4 角色回应 小星陪伴、吉塔稳定情绪、罗素拆解问题。 点击继续回复，或切换角色。 回复结构发生变化，学生能感知IP差异。 高保真
B5 轻量记忆 “要不要让我记住：考试前容易紧张？” 选择“可以记住 / 这次不记 / 管理记忆”。 给用户控制感，避免被偷偷记录的感觉。 中高保真
B6 安全边界 温和边界说明，不用冷冰冰免责声明。 点击“知道了”或“了解什么情况会找支持”。 明确AI不是诊断/咨询，危险时优先安全支持。 中高保真
B7 下一步建议 根据对话给行动卡：继续聊、做呼吸、记录、找可信成年人。 选择一个下一步。 进入对应轻任务或结束反馈。 高保真
B8 对话结束反馈 “这次聊天已记成今天的一小步”，可回首页。 回首页/查看成长记录。 完成一次AI陪伴闭环。 中高保真

Flow B 页面清单

页面 AI对话怎么接 是否必须做
全局AI入口在首页、完成反馈、成长记录页均可唤起；不只依赖底部Tab。必须
三IP选择按用户意图选人：陪伴、稳定、分析。必须
对话页提供预设话题和输入框；Demo用脚本模拟。必须
记忆确认只记轻量信息，必须可关闭和管理。必须
安全边界提示用支持性语言说明AI能力、隐私和风险处理。必须
下一步建议每次对话后给一个具体下一步，不让聊天悬空。必须

页面节点说明

节点 用户看到什么 用户点什么 系统反馈什么 保真度
C1 任务完成 “你完成了一次 2 分钟专注/呼吸练习”。 点击“查看结果”。 进入结果解释。 中保真
C2 结果解释 低压力解释：练习了专注/放松/表达，不出现排名和诊断。 点击“记到成长记录”。 生成成长记录条目。 高保真
C3 奖励获得 “因为完成一次放松练习，获得 3 颗放松星”。 点击“知道了”或“看看能做什么”。 说明奖励去向；可假跳转。 中高保真
C4 成长记录 时间线条目：今天状态、完成行动、AI回应。 点击“让小星回应一下”。 进入 AI 回应。 高保真
C5 AI回应 小星/吉塔说“我看见你刚才完成了”。 点击“回首页”或“继续聊”。 回流首页或进入 Flow B。 高保真
C6 下一次提示 “明天如果还想来，做 1 分钟也可以”。 点击“明天提醒我”或“先不用”。 假提醒状态，不接系统推送。 低中保真
C7 首页回流 首页显示“今天完成了一次放松练习”。 点击成长记录或继续聊。 展示跨模块回流成立。 高保真
C8 奖励去向 星星用途提示：装扮、收藏、星球小物。 点击只展示预览。 假跳转，不做完整商店。 低保真/假跳转

Flow C 页面清单

页面 完成反馈怎么呈现 是否必须做
完成页先确认完成行为，不给成绩压力。必须
结果解释解释“练到了什么能力/状态”，不排名、不诊断。必须
奖励获得说明奖励原因和去向。建议
成长记录记录今天的一小步，连接情绪、行动、反馈。必须
AI回应由 IP 承接完成动作，形成被看见感。必须
奖励去向/商店预览只做假跳转预览，不做完整交易链路。可选

关键页面文案合集

情绪打卡文案

首页主文案今天先从哪里开始？可以先选一个最像现在的状态。
状态词还好 / 有点烦 / 累 / 紧张 / 开心 / 说不清
补充输入提示想补一句也可以，不想说清楚也没关系。

AI回应文案

小星听起来你今天有点烦。可以不用马上说清楚，我先陪你把这一下放慢。
吉塔你现在好像有点绷。我们先做一个很短的呼吸，让身体先松一点。
罗素如果事情有点乱，我们可以先拆成两小块，只处理现在能处理的一步。

三IP角色选择文案

小星我想被陪陪，慢慢说也可以。
吉塔我现在有点难受，想先稳住。
罗素我想把问题理清楚，找到下一步。

安全边界文案

温和说明我会尊重你的分享，也会尽力陪你整理想法。但我不能替代老师、家长或专业帮助。
风险边界如果你可能伤害自己或别人，我会优先帮你找到更安全的支持。
记忆控制这件事要让我记住吗？你之后也可以随时删除。

完成反馈文案

完成确认你完成了一次 2 分钟的小练习。
成长解释这次练习帮你把注意力放回当下，不需要做得完美，完成就已经算数。
奖励说明因为完成了一次放松练习，你获得了 3 颗放松星。
AI回应我看见你刚才做完了。今天已经有一小步被记录下来了。

首页回流文案

已打卡状态今天已记录：有点烦，但你已经做了一次小小的照顾。
继续入口要不要继续和小星聊两句？
轻提醒不用每天都很有状态，愿意回来看看就很好。

Demo原型结构

原型分区 包含页面 保真度 交互要求
00 启动与首页 A1 Pad首页中枢、A8 首页回流、B1 全局AI入口状态、C7 完成回流状态。 高保真 首页必须能进入 Flow A/B；Pad 空间需要展示状态、AI、推荐、成长和次级入口，状态变化要明显。

04 完成反馈 C1-C6、C8。 中高保真，可降级 若时间紧，只做 C2/C4/C5/C7。
99 假跳转 完整商店、完整脑之岛、完整成长报告、完整设置页。 低保真/假跳转 点击后出现“本次demo暂不展开”的过渡页，避免断点。

高保真 / 中低保真 / 假跳转划分

类型 页面 为什么
高保真 Pad 首页中枢、情绪状态选择、AI接住、今日推荐分流、三IP选择、AI对话、边界提示、首页回流。 这是 9 月测试最关键的理解和信任节点。
中高保真 今日小行动、完成页、结果解释、成长记录、AI回应。 需要验证反馈闭环，但视觉复杂度可以低于主入口。
低保真 行动过程、提醒设置、奖励去向预览。 只要支撑路径，不是本轮判断重点。
假跳转 完整商店、装扮系统、完整脑之岛地图、完整报告页、完整设置页。 本轮不验证完整系统，避免范围失控。
