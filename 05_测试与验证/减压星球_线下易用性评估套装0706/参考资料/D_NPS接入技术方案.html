NPS接入方案 · 4个核心触点

每个触点包含三个维度：埋点位置（产品内具体页面与触发时机）、触发逻辑（频次控制与条件判断）、问题内容（主问题 + 条件追问）。

T1
情绪打卡完成 · /clockIn 完成页
首次打卡 · 每7次触发
埋点位置

完形填空第4步（句子生成页）用户点击「完成打卡」→ 成功动画播完 → 结果卡片出现时注入问卷弹层。

即 signStatus 由 1/2 变为 3/4 的状态切换完成后触发 · 非打卡进行中触发

触发逻辑
  • 首次完成打卡后立即触发
  • 之后：累计打卡次数达 7 的倍数时触发
  • 同一自然日内已触发过其他NPS → 跳过
  • 用户关闭问卷 → 标记「已拒绝」，30天内不再触发 T1
问题内容
📔 今天记录心情的过程，感觉怎么样？
😄 很顺畅 😐 还好 😕 有点卡
若选「有点卡」→ 追问：「哪里卡住了？」
（找不到合适的情绪词 / 事件选项太少 / 句子不像我说的 / 其他）
T2
AI对话报告生成 · /conversation/report
首次报告 · 每14天一次
埋点位置

对话报告页面加载完成、报告卡片完全展示后，延迟 2秒 触发弹层。

仅在报告成功生成后触发（至少完成一问一答）· 三IP均计入（吉塔/小星/第三IP） · 对话 < 2轮不触发

触发逻辑
  • 首次生成对话报告后立即触发
  • 之后：距上次T2触发满 14天 且有新报告生成
  • 仅发问题未获AI回复即退出 → 无报告，不触发
  • 30天内已拒绝 → 不触发
问题内容
💬 和AI伙伴聊完，你感觉怎么样？
😊 好多了 🙂 还行 😔 没什么感觉
若选「没什么感觉」→ 追问：「是因为什么？」
（AI回复太短 / 感觉没被理解 / 我不知道说什么 / 其他）
T3
脑之岛认知训练完成 · 训练结束页
首次训练 · 每30天一次
埋点位置

训练结束动画播完 → 勋章获得弹窗出现 → 用户点击关闭勋章弹窗后，注入NPS弹层。

仅在正式训练完成后触发，「下载资源」进度页不触发 · gradeCode权限未开放导致无法完成的不触发

触发逻辑
  • 首次完成任意岛屿任意训练后触发
  • 之后：距上次T3触发满 30天
  • 4岛均计入（注意力/记忆力/思维力/执行力）
  • 训练中途退出（未完成） → 不触发
  • 30天内已拒绝 → 不触发
问题内容
🧠 刚才的训练，你觉得怎么样？
🔥 很有趣 😐 还行 😑 有点无聊
若选「很有趣」→ 追问：「想继续哪个方向？」（注意力 / 记忆力 / 思维力 / 执行力）
若选「有点无聊」→ 追问：「是什么感觉？」（太难 / 太简单 / 加载太慢 / 不知道有什么用）
T4
整体满意度 · 首页任务完成弹窗 / 持久横幅
安装满14天 · 每90天一次
埋点位置

两种入口（优先级从高到低）：

  • 优先：任务系统完成当日全部任务的弹窗关闭后触发
  • 备用：首页启动时底部插入持久横幅（可滑走，不阻断操作）

同日已触发备用横幅不重复触发

触发逻辑
  • 首次触发：安装满14天 AND 累计使用天数 ≥ 5天
  • 之后：距上次T4触发满 90天
  • 同周期内已触发T1/T2/T3 → 延迟 3天 再触发T4
  • 仅当日首次打开App触发（不在使用中途弹出）
